1.1 Simple Installation

Tatami is released as a single Jar (tatami.jar) which contains all the resources necessary: code Java, files of configurations, files CSS, images and JavaScript code. To use Tatami, it is enough to add this jar in the classpath of your application and to insert in the file of configuration of your module (.gwt.xml) the following line:

<inherits name="com.objetdirect.tatami.Tatami"/>

You do not have to choose import the dojo stylesheets, nor add the corresponding class in your markup: the theme package takes care of that for yourself!
